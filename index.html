Thank you for choosing Mobile Client Kiosk!

You need to distribute the app using a Mobile Device Management (MDM) solution and implement Manage App Configuration to customize settings.

Also, the app leverages iOS autonomous Single App Mode, which can be configured with a configuration profile with Restrictions payload.
