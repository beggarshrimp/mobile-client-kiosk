Autonomous Single App Mode

To use the Autonomous Single App Mode feature in the app, you need to install a configuration profile on the device using an MDM solution.

This profile must contain the following payload:

<key>autonomousSingleAppModePermittedAppIDs</key>
<array>
    <string>ru.yandex.mobile-client-invite-kiosk</string>
</array>

To exit Single App Mode swipe right three times on the screen and enter the PIN code, which should also be set up using the Managed App Configuration through your MDM solution
